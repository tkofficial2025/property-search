# 路線ロゴ画像

このフォルダには、日本の鉄道路線の公式ロゴ画像を配置してください。

## 必要なロゴファイル

以下のファイル名で公式ロゴ画像を配置してください：

### JR東日本
- `jr-east.jpg` または `jr-east.png`（使用しない - 色付きアイコンで表示）

### 東京メトロ
- `tokyo-metro-g.jpg` または `tokyo-metro-g.png` - 銀座線（Ginza Line）
- `tokyo-metro-m.jpg` または `tokyo-metro-m.png` - 丸ノ内線（Marunouchi Line）
- `tokyo-metro-h.jpg` または `tokyo-metro-h.png` - 日比谷線（Hibiya Line）
- `tokyo-metro-t.jpg` または `tokyo-metro-t.png` - 東西線（Tozai Line）
- `tokyo-metro-c.jpg` または `tokyo-metro-c.png` - 千代田線（Chiyoda Line）
- `tokyo-metro-z.jpg` または `tokyo-metro-z.png` - 半蔵門線（Hanzomon Line）
- `tokyo-metro-n.jpg` または `tokyo-metro-n.png` - 南北線（Namboku Line）
- `tokyo-metro-y.jpg` または `tokyo-metro-y.png` - 有楽町線（Yurakucho Line）
- `tokyo-metro-f.jpg` または `tokyo-metro-f.png` - 副都心線（Fukutoshin Line）

### 都営地下鉄
- `toei-a.jpg` または `toei-a.png` - 浅草線（Asakusa Line）
- `toei-i.jpg` または `toei-i.png` - 三田線（Mita Line）
- `toei-s.jpg` または `toei-s.png` - 新宿線（Shinjuku Line）
- `toei-e.jpg` または `toei-e.png` - 大江戸線（Oedo Line）

## ロゴの取得方法

公式ロゴは以下のサイトから取得できます：

1. **東京メトロ**: https://www.tokyometro.jp/
2. **都営地下鉄**: https://www.kotsu.metro.tokyo.jp/
3. **JR東日本**: https://www.jreast.co.jp/

ロゴを使用する際は、各社の利用規約を確認してください。

## フォールバック

ロゴ画像が存在しない場合、自動的に色付きの円形アイコンが表示されます。
