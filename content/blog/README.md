# ブログ記事

このフォルダにMarkdownファイルとして記事を保存してください。

## ファイル名の形式

`YYYY-MM-DD-slug.md`

例:
- `2024-01-15-tokyo-expat-housing-guide.md`
- `2024-01-20-investment-tips.md`

## ファイルの構造

```markdown
---
title: "記事のタイトル"
date: "2024-01-15"
category: "Guide"
featuredImage: "/blog-images/tokyo-expat-housing.jpg"
---

# 記事のタイトル

記事の本文をここに書きます...
```
